Guess the starting XI of historic football matches using Wordle-style clues

## Gameplay Mechanics

1.  **Selection:** A player clicks an empty slot (e.g., 'GK' or 'CM') on the pitch.
2.  **Guessing:** A search bar appears, allowing the user to guess a player's name.
3.  **Goal:** Identify all 11 players before running out of guesses for any single position. The **Kit Number** is provided as the default hint for every position.
